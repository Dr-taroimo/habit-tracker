# 習慣トラッカー

毎日の小さな習慣を、気軽に記録できるトラッカーです。

「朝に水を飲む」「ストレッチする」「本を読む」など、続けたいことを登録して、できた日にチェックをつけられます。直近7日間の達成状況や連続日数が見えるので、無理なく続けるきっかけになります。

## 主な機能

- 習慣の追加
- 今日できた習慣のチェック
- 達成の取り消し
- 連続日数の表示
- 直近7日間の達成状況
- 今週の達成数
- 効果音のオン/オフ
- 習慣ごとの通知時刻の設定

## 使い方

1. 続けたい習慣を入力します。
2. `習慣を追加` を押します。
3. 今日できたら `今日できた` を押します。
4. 間違えて押したときは `取り消す` で戻せます。

毎日少しずつチェックしていくと、連続日数と直近7日間の記録が更新されます。

## データについて

入力した習慣や達成記録は、利用している端末やブラウザ内に保存されます。外部サーバーへ送信する仕組みはありません。

## 通知について

通知を許可すると、習慣ごとに設定した時刻にチェックを思い出しやすくなります。ブラウザ版では、ページを開いている間に通知が動作します。
